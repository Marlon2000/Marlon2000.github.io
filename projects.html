Object Oriented Design

Uno Card Game

This card game is written in the object-oriented based language Java. A detailed planning and a good implementation had to be done, in order to get everything working fine. All the classes have to interact with each other and all the cards need to be properly displayed. So it took me a long time to find out how all the different classes should interact with each other. Also I had to consider all the rules, which I had to implement afterwards. But now as it is done I am quite happy with the overall result.
